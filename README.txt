GitTop - A Lightweight GitHub Notifications Client
==================================================

Why spin up a browser just to check your GitHub notifications?

QUICK START
-----------
1. Open GitTop
2. Add your GitHub account via Settings > Accounts
3. Optionally enable Power Mode via Settings > Power Mode
4. Optionally configure rules via Rule Engine Settings > Rules
5. Done! You'll receive notifications in your system tray

FEATURES
--------
- Super lean: ~5-15MB RAM in use, 1-2MB in tray
- Multi-account support
- Smart rules engine for priorities and filtering
- Cross-platform: Windows and Linux
- Dual mode: Minimalist or Power Mode

CONFIGURATION
-------------
Settings are stored in:
  Windows: %APPDATA%\GitTop
  Linux:   ~/.config/GitTop
